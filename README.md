# 💊 DrugDose

**DrugDose** è un'applicazione Android sviluppata come progetto per il corso di **Programmazione di Dispositivi Mobili**.

L'obiettivo dell'applicazione è fornire uno strumento digitale a supporto dei medici nella gestione dei farmaci e nella creazione di prescrizioni, semplificando la ricerca dei medicinali e organizzando il processo di prescrizione attraverso un'interfaccia intuitiva.

Il progetto è stato sviluppato interamente in **Kotlin** utilizzando le moderne tecnologie Android.

---

# 📱 Funzionalità principali

## 🔎 Ricerca farmaci

L'applicazione permette di:

- Cercare farmaci tramite un sistema di ricerca dedicato
- Visualizzare le informazioni principali del medicinale
- Consultare il documento ufficiale RCP (Riassunto delle Caratteristiche del Prodotto)
- Selezionare un farmaco per iniziare la creazione di una prescrizione

---

## 📝 Creazione prescrizioni

È possibile creare una prescrizione attraverso un processo guidato:

- Inserimento dei dati del paziente
- Selezione del farmaco
- Configurazione della quantità e del numero di confezioni
- Inserimento delle informazioni relative alla terapia
- Generazione della prescrizione

---

## 👨‍⚕️ Gestione dati medico/paziente

L'applicazione è progettata per supportare il flusso di lavoro del medico permettendo:

- Organizzazione delle informazioni dei pazienti
- Consultazione rapida dei farmaci
- Gestione semplificata delle prescrizioni

# 🛠️ Tecnologie utilizzate

| Tecnologia | Utilizzo |
|---|---|
| **Kotlin** | Linguaggio principale |
| **Android SDK** | Framework per lo sviluppo mobile |
| **Jetpack Compose** | Realizzazione dell'interfaccia grafica |
| **MVVM** | Architettura dell'applicazione |
| **ViewModel** | Gestione dello stato UI |
| **StateFlow** | Aggiornamento reattivo dei dati |
| **Firebase** | Database e servizi backend |
| **Gradle Kotlin DSL** | Gestione del progetto |
